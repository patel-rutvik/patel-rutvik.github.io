Periodic Table Database

A database for elements in the periodic table where the user can browse through all the elements, with different UI layouts available to use. This project also has predictive search, using the Trie data structure. Ecrypted Arduino Communication

This program (once downloaded onto both Arduinos) allows for the generation and exchange of private and public keys, establishing a secure connection between the two devices. Each character is then encrypted using a sequence of randomly generated keys, allowing for each character to encrypted using a different key than the last. These bits are then transmitted to the other device, where it is then decrypted using the same process.

Image Compressor using Huffman Trees

This program demonstrates the implementation of the "greedy" huffman algorithm. To achieve this, we compress each bit using the huffman tree, and also compress the huffman tree itself using the pickle module. It is then decompressed by walking up the tree and displayed on a local host on the user's computer.
